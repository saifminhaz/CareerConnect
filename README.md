# CareerConnect

CareerConnect is a professional networking platform designed to help users connect with colleagues, find jobs, and manage their professional identity. It mimics the core functionality of major professional networking sites with a focus on a clean, modern user experience.

## Features

-   **Home Feed**: View posts from your network and followed pages.
-   **Profile Management**: customizable user profiles with headlines, stats, and activity.
-   **Network**: Manage connections and view potential contacts.
-   **Jobs**: Browse and apply for job opportunities with tracking.
-   **Messaging**: Real-time messaging interface (simulation).
-   **Notifications**: Stay updated with alerts and interactions.
-   **Tracker**: A dedicated application tracker to monitor job applications.
-   **Responsive Design**: Works seamlessly on desktop and mobile devices.

## Technologies Used

-   **HTML5**: Semantic structure.
-   **CSS3**: Custom styling with Flexbox and Grid, using CSS variables for theming.
-   **JavaScript (ES6+)**: Interactive logic for modals, navigation, and state simulation.
-   **Google Fonts**: Inter font for modern typography.

## Setup & Installation

To run this project locally:

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    ```
2.  Navigate to the project directory:
    ```bash
    cd careerconnect
    ```
3.  Open `index.html` in your preferred web browser.

    *Optional: Use a local server (like Live Server in VS Code) for the best experience.*

## Deployment

This project is configured for deployment on Surge.sh.

1.  Ensure you have `surge` installed globally or use `npx`.
2.  Run the deployment script:
    ```bash
    ./deploy.sh
    ```

## Project Structure

-   `index.html`: Main landing page / Home feed.
-   `profile.html`: User profile page.
-   `network.html`: Network management page.
-   `jobs.html`: Job search and listing page.
-   `tracker.html`: Job application tracker.
-   `messaging.html`: Messaging interface.
-   `notifications.html`: Notifications page.
-   `style.css`: Global stylesheets.
-   `script.js`: Shared JavaScript logic.
-   `deploy.sh`: Deployment automation script.
-   `assets/`: Images and other static resources (e.g., `profile_pic.png`, `cover_photo.png`).
